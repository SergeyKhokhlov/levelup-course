Компьютерная программа — последовательность инструкций, предназначенная для исполнения устройством управления вычислительной машины. Программа — один из компонентов программного обеспечения. В зависимости от контекста, рассматриваемый термин может относиться также и к исходным текстам программы. Компьютерные программы, как объект авторского права и других прав интеллектуальной собственности, относится к категории нематериальных активов.

У вас на компьютере много разных программ. Например: программа блокнот позволяет вам редактировать тексты, программа-браузер находит и показывает вам страницы в интернете.

Есть разные способы записать программу. Но наиболее «удобная» для компьютера форма записи в виде машинного кода, неудобна для человека. Поэтому для составления программ придумали языки программирования.

Язык программирования

Язык программи́рования — формальный язык, предназначенный для записи компьютерных программ.

Давайте перейдем непосредственно к языку программирования python.

У данного языка есть 2 варианта произношения - "Питон" или "Пайтон".

Python относится к интерпретируемым языкам программирования, чтобы запустить написанную на Python программу, нужен интерпретатор Python (я использую PyChram, скачать его можно по этой ссылке: https://www.jetbrains.com/ru-ru/products.html).
⠀

А теперь давайте изучим первую нашу команду. Команда print().

Команда print() используется для вывода текста в консоль.
⠀
print("Привет")
>> Привет
⠀
Давайте создадим файл с расширением .py и запишем в него следующую строчку: print("25.07.2020). Далее нажмем на зеленую кнопку и запустим нашу программу. Если вы всё сделали правильно, то в консоли должна появится следующая строчка: 25.07.2020
⠀
Следующая немаловажная тема - считывание текста/числа. Для этого нужна команда input().

Например мы хотим написать программу, которая будет отвечать на каждый вопрос словом "да".

Для начала нам нужно сделать считывание строки, куда пользователь будет писать свой вопрос. Затем нужно вывести в консоль слово "да".

Данная программа будет выглядеть следующим образом:
⠀
line = input()
print("да")

> 15 + 15 = 30?
>> да
⠀
Первая строчка программы это считывание текста в переменную line. Следовательно что бы мы не ввели, всё будет в переменной line. Во-второй строчки программы, мы выводим слово "да", используя изученную нами команду print().

Теперь, давайте напишем программу, которая будет отвечать на вопросы, и в конце выведет в консоль всю информацию, которую мы ввели. 
⠀⠀
print("Как вас зовут"?)  # программа выведет в консоль вопрос: "как вас зовут?".
name = input()  # теперь вы должны ввести ваше имя в консоль
print("Сколько вам лет?") # программа выведет в консоль вопрос: "Сколько вам лет?".
age = input()  # теперь вы должны ввести ваш возраст в консоль
print("Отлично! Вас зовут", name, ", и ваш возраст:", age, "лет.")  # Здесь, мы вывели ваше имя и ваш возраст. Весь текст пишется в ковычках, если же вы хотите вывести значение переменной, то не нужно его брать в ковычки.
⠀
